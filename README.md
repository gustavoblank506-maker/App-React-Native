# Meu Perfil

Aplicativo mobile desenvolvido com React Native e Expo para apresentar um perfil profissional de estudante de forma simples, moderna e funcional.

## Sobre o projeto

O **Meu Perfil** funciona como um cartão digital. Ele reúne informações básicas sobre o aluno, sua linguagem de programação favorita, área de interesse e formas de contato em uma única tela.

O aplicativo foi desenvolvido como atividade prática de introdução ao React Native, utilizando Expo Go para execução no dispositivo móvel.

## Problema que o app resolve

Muitas vezes, estudantes precisam se apresentar de forma rápida, organizada e profissional. Em vez de enviar nome, contato e área de interesse separadamente, este aplicativo centraliza essas informações em uma interface simples e acessível.

## Objetivo

Facilitar a apresentação profissional de estudantes por meio de um cartão digital moderno, com informações pessoais, área de interesse, linguagem favorita e botões de contato.

## Funcionalidades

- Exibição da logo do SENAI
- Tela de splash animada
- Apresentação do perfil do aluno
- Exibição da linguagem favorita
- Informações sobre área de interesse e status de estudo
- Botão de contato pelo WhatsApp
- Botão para GitHub/Portfólio
- Modo claro e modo noturno
- Transição suave entre temas
- StatusBar adaptada ao tema
- Menu clicável em "Meu Perfil"
- Modal de configurações
- Opção de sair do aplicativo

## Tecnologias utilizadas

- React Native
- Expo
- JavaScript
- JSX
- Expo Go
